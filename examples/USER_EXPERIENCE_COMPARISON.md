## 🔄 完整流程对比

### ❌ 旧流程：复杂且容易出错

```bash
# ========== 用户噩梦：8 步手动配置 ==========

# 步骤 1: 创建技能目录
mkdir -p skills/tushare-quant
cd skills/tushare-quant

# 步骤 2: 编写 SKILL.md
# 用户需要学习 YAML frontmatter 格式
cat > SKILL.md << 'EOF'
---
id: tushare-quant
name: Tushare Quant
description: 使用 Tushare API ...
version: 1.0.0
...

# 用户需要编写 100+ 行的技能定义
# 很容易格式错误

# 步骤 3: 运行安全审计
vibe analyze security ../tushare-quant/
# ⚠️ 可能发现安全问题需要修复

# 步骤 4: 安装技能
vibe skills install tushare-quant --source ../tushare-quant/
# ✗ 可能因为格式问题失败

# 步骤 5: 配置 manifest.yaml
# 用户需要理解复杂配置结构
cat >> .vibe/manifest.yaml << 'EOF'
skills:
  project:
    - id: tushare-quant
      priority: ??  # 应该设置多少？
      enabled: true
EOF

# 步骤 6: 配置路由规则
# 用户需要懂正则表达式
cat >> .vibe/policies/task-routing.yaml << 'EOF'
rules:
  - pattern: ".*(tushare|股票).*"  # 正则对吗？
    skill: tushare-quant
    mode: ??  # mandatory 还是 suggest？
EOF

# 步骤 7: 验证安装
vibe skills verify tushare-quant
# ⚠️ 可能发现配置问题

# 步骤 8: 同步到平台
vibe build --platform claude-code
# ⚠️ 可能因为配置错误失败

# ========== 结果：耗时 30-60 分钟，容易出错 ==========
```

**痛点**：
- ❌ 需要学习 YAML 格式
- ❌ 需要理解配置结构
- ❌ 需要懂正则表达式
- ❌ 不知道优先级设置多少
- ❌ 容易出现格式错误
- ❌ 错误信息不清晰
- ❌ 修复困难

## 🤖 AI 智能配置示例

### 用户输入

```yaml
技能描述: "使用 Tushare API 开发量化交易策略"
触发条件: "用户需要获取股票数据、开发量化策略或回测交易策略"
```

### AI 自动生成

```yaml
# .vibe/skills/auto-config.yaml
skills:
  tushare-quant:
    skill_id: tushare-quant
    priority: 70              # ← AI 根据类别自动设定
    enabled: true
    scope: project
    category: development      # ← AI 分析得出
    tags:                      # ← AI 提取关键词
      - quant
      - finance
      - trading
      - api
    routing:
      patterns:                # ← AI 自动生成正则
        - .*tushare.*
        - .*股票.*
        - .*量化.*
        - .*策略.*回测
      priority: 70
    metadata:
      auto_configured: true
      confidence: 0.92         # ← AI 置信度
      enhanced_at: "2025-04-20T10:30:00"
```

**AI 决策过程**：
1. 分析技能描述 → 类别：development
2. 匹配历史数据 → 优先级：70
3. 提取关键词 → 标签：quant, finance, trading
4. 生成路由模式 → 正则：.*tushare|股票|量化.*
5. 风险评估 → 安全等级：Safe

## 📦 .skill 文件格式

### 标准化分发

```bash
# 之前：需要自己编写和打包
mkdir my-skill
# ... 编写 SKILL.md
# ... 编写配置
tar -czf my-skill.skill ...

# 现在：使用标准格式
vibe skill create my-skill \
  --description "我的技能" \
  --tags "tag1,tag2"

# 或从官方仓库直接安装
vibe skill add tushare
```

### 文件结构

```
tushare.skill (tar.gz)
```
