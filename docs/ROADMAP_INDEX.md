## 🤝 贡献指南

### 如何更新文档

1. **选择合适的文档**
   - 重大变更 → 更新 [完整路线图](./roadmap-2.0.md)
   - 进度更新 → 更新 [项目看板](./roadmap-2.0-board.md)
   - 时间调整 → 更新 [开发时间表](./roadmap-2.0-timeline.md)

2. **保持一致性**
   - 更新所有相关文档
   - 同步更新版本号
   - 记录变更历史

3. **审查流程**
   - 技术内容 → 技术负责人审查
   - 进度更新 → 项目经理审查
   - 重大变更 → 团队讨论
