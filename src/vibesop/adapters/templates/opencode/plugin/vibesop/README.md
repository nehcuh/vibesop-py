# VibeSOP Plugin for OpenCode

> **Reference Implementation** — OpenCode plugin APIs are experimental.

## Overview

This plugin provides automatic skill routing and context injection for OpenCode,
mirroring the behavior of Claude Code's hook-based integration.

## Features

- **Auto-routing**: Intercepts user messages and triggers `vibe route`
- **Skill injection**: Injects matched skill content into system prompt
- **Multi-intent support**: Detects complex queries and generates execution plans
- **Tool tracking**: Records tool usage for session context

## Installation (Future)

```bash
# Generated by: vibe build opencode --with-plugin
cp -r plugin/vibesop ~/.opencode/plugin/
```

## Plugin Hooks

| Hook | Purpose |
|------|---------|
| `experimental.chat.system.transform` | Inject active skill/plan into system prompt |
| `chat.message` | Trigger routing on user messages |
| `tool.execute.before` | Track tool usage |

## Architecture

```
User Message
    ↓
chat.message hook → vibe route --json
    ↓
┌─────────────┬─────────────┐
↓             ↓             ↓
Single Skill  Multi-Intent  No Match
    ↓             ↓             ↓
setActiveSkill  setActivePlan   Toast: fallback
    ↓             ↓
system.transform injects content on next turn
```
